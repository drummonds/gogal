# Example 01a — Axis Formats

Visual test of axis tick formatting across different data ranges.

## What it demonstrates

A gallery of charts showing how gogal formats axis ticks for:

- **Numeric**: small integers, decimals, hundreds, large ranges, negative ranges, narrow ranges
- **Scientific**: very small (1e-6) and very large (1e9) values
- **Temporal**: hours, days, months, years
- **Ordinal**: weekday labels, named categories

## Running

```
task example:01a
```

Opens at [http://localhost:1339](http://localhost:1339).
